# Dashboard de Vendas em Excel | Máxi Pastéis

## Sobre o Projeto

Projeto de análise de dados desenvolvido em Microsoft Excel com foco na construção de um dashboard executivo para monitoramento de performance comercial de uma operação de vendas simulada da empresa fictícia **Máxi Pastéis**.

O objetivo do projeto foi transformar uma base de dados bruta em um painel interativo de apoio à decisão, consolidando indicadores de desempenho e análises visuais para acompanhamento de faturamento, lucratividade, mix de produtos e evolução temporal das vendas.

---

## Objetivos de Negócio

* Monitorar o desempenho comercial da operação de vendas.
* Identificar produtos de maior faturamento e volume.
* Avaliar lucratividade e margem bruta consolidada.
* Acompanhar evolução mensal de receitas.
* Permitir análise dinâmica por filtros de período, produto e categoria.

---

## Principais Análises Disponíveis

### Performance por Produto

Visualização de faturamento, lucro e volume de vendas por item do portfólio.

### Análise de Mix de Categoria

Distribuição percentual de faturamento por categoria de produto.

### Tendência Temporal

Acompanhamento do faturamento ao longo dos meses para identificação de sazonalidade e performance.

### Ranking de Produtos

Top 3 produtos mais vendidos com destaque visual.
